## What?

- Code structure for a normal python flask application

  Includes:

  - One blueprint route - `/`
  - Templates
  - Model class with mysql python migrate


## Setup

```
cd <pathToFolder>/pythonFlaskBoilerplate
virtualenv venv
pip install -r requirements.lock
python run.py
```

## Useful commands

- `flask db migrate`
- `flask db upgrade`
